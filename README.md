# UdaanX University Event Portal — FIXED VERSION

## What was wrong (root cause)
The previous version used libraries that were NOT installed:
  ❌ flask-sqlalchemy  → replaced with sqlite3 (built into Python)
  ❌ flask-bcrypt      → replaced with werkzeug.security (comes with Flask)
  ❌ flask-limiter     → replaced with a simple stdlib rate limiter
  ❌ qrcode            → replaced with PIL image generation
  ❌ bleach            → replaced with regex-based tag stripper
  ❌ flask-caching     → removed (not needed for this scale)

## Libraries used NOW (all confirmed working)
  ✅ flask             (pip install flask)
  ✅ pandas            (pip install pandas)
  ✅ matplotlib        (pip install matplotlib)
  ✅ Pillow            (pip install Pillow)
  ✅ sqlite3           (built into Python - no install)
  ✅ werkzeug          (comes with Flask - no install)

## How to run
Step 1 — Install packages (only 4 needed):
  pip install flask pandas matplotlib Pillow

Step 2 — Run:
  python app.py

Step 3 — Open browser:
  http://127.0.0.1:5000

## Admin Dashboard
